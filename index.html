Entirely built from scratch:

This project's aim was to learn the "backend" of how login and registration pages work as well as sessions and building RESTful architecture from scratch. It contains:

  • a full user registration / login with validation
  • Facebook's "Wall"
  • Evernote's "notes"

Stack used:

  • PHP
  • jQuery/AJAX (little)
  • MySQL
  • Twitter Bootstrap
